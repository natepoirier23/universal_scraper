# Book Scraper (Demo Project)

This project demonstrates a simple web scraper using Python.

It collects book data from the first 5 pages of  
https://books.toscrape.com/

## Extracted fields

- title
- price
- rating
- url

## Tech

- Python 3
- requests
- beautifulsoup4
- csv

## How to run

```bash
pip install -r requirements.txt
python scrape_books.py
